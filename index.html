MacOS搭建本地开发环境

在Mac机上安装下面两个软件

  1. VirtualBox
  2. Vagrant

···

安装vagrant的插件

VirtualBox 5.1.22 以后的版本的话会有错误出现，不要执行下面的命令

vagrant plugin install vagrant-vbguest

先移动到主目录

cd

创建虚拟机文件的目录文件夹：MyVagrant

mkdir MyVagrant

转到虚拟文件夹中

cd MyVagrant

创建一个名为MyCentOS的虚拟子文件夹

mkdir MyCentOS

移动到虚拟子文件夹中去

cd MyCentOS

创建初始化文件Vagrantfile

vagrant init bento/centos-6.8

编辑Vagrantfile，设定虚拟机的IP为192.168.33.10

sed -i ‘’ -e ’s/# config.vm.network “private_network”, ip: “192.168.33.10”/config.vm.network “private_network”, ip: “192.168.33.10”/‘ Vagrantfile

启动虚拟机

vagrant up

确认虚拟机的状态

vagrant status ···
